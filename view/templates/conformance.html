Conformance

This API conforms to the following conformance classes:

  • Open API Core
  • OpenAPI 3.0
  • OpenAPI HTML

Not that this API does not conform to OpenAPI GeoJSON since the geometry of its spatial features doesn't use a WGS84 or related coordinate reference system, instead it uses the TB16Pix DGGS.
